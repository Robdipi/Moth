# Moth
A cool locking collection of small "edgy" locking terminal tools writen in Jave.


It is modular build and uses a MVC+ Architecture. with a Terminal interface at its core.
It has a bunch of commands that it can run when it runs with flags and user input. 
like:
- shutdown - closes the app
- help - gives a list of all commands
- imgen - will implement later has a field for a filename and a password
- imgde - will implement later has a field for a filename and a password
